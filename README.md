# TaskFlow — Project Management Tool

A Trello-style full-stack project management tool.

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, JWT, MongoDB
- **Frontend:** React.js
- **Database:** MongoDB Atlas (Cloud)

## Features
- JWT Authentication
- Create projects and assign team members
- Drag-and-drop Kanban board (To Do / In Progress / Done)
- Automated email reminders for upcoming deadlines
- PDF export for task summaries

## How to Run
### Backend
mvn spring-boot:run

### Frontend
npm install
npm start
